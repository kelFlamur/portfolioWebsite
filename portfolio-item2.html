A Spring Boot Project for Employee Management

Using React + Spring Boot

The Point of this project was to show the four basic CRUD commands using Spring Boot - a microservice. In the application you are able to add employees by First Name, Last Name, and Email. You are also able to delete an employee and update one as well. The Employee Managment App data is displayed in a table so it is easier to view all employees.

The app uses React for the front end and Spring Boot for the back end. For communication between the back end and front end I used the axios library. I also used Bootstrap for a lot of the styling. Finnaly I used a MySQL database to store all the data and create the proper fields. Spring Boot was helpful for this because I untitilzed a libary in Spring called Hibernate to automatically generate all the fields for me when I create them in the back end.
